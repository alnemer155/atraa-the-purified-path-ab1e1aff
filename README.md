# عِتَرَةً — Atraa

منصة إسلامية شيعية شاملة. تطبيق عربي (RTL) مع دعم للإنجليزية، مُحسَّن للهواتف.

## المميزات
- أوقات الصلاة (دعم مدن العالم باستثناء فلسطين المحتلة)
- اتجاه القبلة
- مكتبة الأدعية والزيارات والأذكار والقرآن الكريم
- التسبيح الرقمي
- التقويم الهجري ومناسبات أهل البيت عليهم السلام

## التطوير
```bash
npm install
npm run dev
```

## النشر للجوال
يستخدم المشروع Capacitor لإنتاج تطبيقات iOS و Android.
