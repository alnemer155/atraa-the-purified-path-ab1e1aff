## ✨ المميزات الأساسية / Core features

- 🕌 **أوقات الصلاة** — حساب جعفري عبر AlAdhan API (دعم مدن العالم باستثناء فلسطين المحتلة وتايوان).
- 🧭 **بوصلة القبلة** — حساب دائرة كبرى دقيق + فلتر دائري للبوصلة + اهتزاز عند المحاذاة.
- 📖 **القرآن الكريم** — رسم عثماني، تشغيل صوتي بقُرّاء متعدّدين، علامات مرجعية.
- 🤲 **مكتبة الأدعية والزيارات والأذكار** — مع دعم محتوى المذهبَين (شيعي/سني).
- 📿 **التسبيح الرقمي** + التقويم الهجري ومناسبات أهل البيت ﷺ.
- 🎨 **تصميم بسيط أبيض/أسود** — بلا إيموجي، بخط IBM Plex Sans Arabic.
- 🌙 **وضع ليلي/سيبيا** لقراءة القرآن.

## 🛡️ الأمان والخصوصية / Security

- لا تخزين بيانات شخصية على الخادم بدون موافقة (Lovable Cloud + RLS).
- جميع الطلبات الخارجية HTTPS فقط (`upgrade-insecure-requests`).
- منع التضمين في `<iframe>` خارجية (`X-Frame-Options: DENY`).
- منع تتبّع الكوهورت الإعلاني (`interest-cohort=()`).
- تشفير كلمات المرور وOTP عبر Supabase Auth (٦ أرقام إلزامي).

تفاصيل كاملة في `index.html` و `vercel.json` و `COPYRIGHT.txt`.

## 📱 النشر للجوال / Mobile deployment

يستخدم المشروع **Capacitor** لإنتاج تطبيقات iOS و Android.

```bash
npm run build
npx cap sync
npx cap run ios      # macOS + Xcode
npx cap run android  # Android Studio
```

> ⚠️ **قبل بناء الإنتاج**: احذف بلوك `server` من `capacitor.config.ts`.
